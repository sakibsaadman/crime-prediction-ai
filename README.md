# Crime Prediction using Artificial Intelligence

This project aims to develop a machine learning system to predict crime type, time, and location in West Yorkshire using historical open crime data.

## Dataset
Source: https://data.police.uk/

Datasets used:
- 2025-08-west-yorkshire-street.csv  
- 2025-08-west-yorkshire-outcomes.csv  

## Project Structure
- data/raw: Raw datasets from data.police.uk  
- data/processed: Cleaned and feature-engineered datasets  
- notebooks: Jupyter notebooks for data exploration and modelling  
- src: Python scripts for preprocessing and feature engineering  
- docs: Project documentation (charter, Scrum plan, reports)
